Hangman Game

A terminal hangman game made in C.

Do you remember those hangman games we used to play in English class?
We decided to make it into a terminal game, in a style similar to those early 80s computer games.

คุณจำเกม hangman ที่ชอบมีการเล่นในคาบเรียนภาษาอังกฤษได้ไหม
เราได้นำเกมนั้นมาสร้างเป็นเกม terminal ใน style คล้ายเกมยุค80



How to play?

After installing and compiling to fit your system, playing this game is simple, like the real life hangman played on a board.
After starting the game, you choose a dificulty level for the words you want, then you will be assigned random words to play.
Guess a letter or the entire word, and it will be added to your progress if you're correct, or the hangman will grow limbs and your life will decrease if you're wrong....
You can guess wrong 7 times.

Documentations

This game was made in C using just the standard libraries and requires nothing but the source code, the words.txt files,
and a C compiler for your system to compile and run.

The game starts by clearing the screen and allocating spaces for all the texts that will be use, shows are a welcome screen, set the difficulty, get the words, and start the game loop.
Each of the functions serve a specific purpose, which are:
main() The main fucntion that runs the program.
screenUpdate() Updates the output on the screen.
welcome_screen()> Create a welcome screen to greet the player and choose the difficulty level.
setDifficulty() Open the words.txt files according to the difficulty choosen.
getWordAndHint() Choose a random word taken from the words.txt files.
createProgressBar() Create a progress bar used to show the letters guessed correctly.
updateProgressBar() Update the progress bar each time the player guessed correctly.
notInProgressBar() Check if a letter has already been guessed correctly.
correctCount() Count the number a letter appear in the choosen word.
findSpace() Find the number of whitespaces in the choosen word.
playGame() The main game loop that takes the input and count the scores.
clearGame() Clear all the memory and filestream.

เกมนี้ถูกสร้างในภาษา C โดยใช้เพียง standard libraries การเรียกใช้เกมนี้ต้องมีไฟล์คำศัพท์ และ คอมไพเลอร์ภาษา C เพื่อ คอมไพล์สำหรับคอมพิวเตอร์คุณ

เมื่อเริ่มเกมจะทำการล้างหน้าจอแล้วเตรียมพื้นที่ในRAMสำหรับเก็บข้อความและข้อมูลที่จะใช้ เปิดหน้าจอต้อนรับ เลือกระดับความยาก รับคำศัพท์จากไฟล์ที่เก็บไว้ แล้วเริ่มต้น loop เกม
แต่ละฟังก์ชันในโค้ดของโปรแกรมมีหน้าที่ของตัวเอง ได้แก่:
main() ฟังก์ชันที่ควบคุมการทำงานของโปรแกรม
screenUpdate() เปลี่ยนการแสดงผลบนหน้าจอ
welcome_screen()> หน้าจอต้อนรับและเลือกระดับความยาก
setDifficulty() นำเข้าคำศัพท์จากไฟล์ตามระดับความยากที่เลือก
getWordAndHint() สุ่มคำศัพท์จากไฟล์ที่นำเข้ามา
createProgressBar() สร้าง progress bar แสดงความยาวของคำและตัวอักษรที่ทายถูก
updateProgressBar() เพิ่มตัวอักษรใน progress bar
notInProgressBar() ตรวจดูว่าตัวอักษรถูกตอบไปแล้วหรือยัง
correctCount() นับจำนวนตัวอักษรที่ทายถูกในคำศัพท์
findSpace() ตรวจหาการเว้นระหว่างคำ (whitespace) ในคำศัพท์
playGame() ฟังก์ชันที่ควบคุมการ input output และนับคะแนนเกม
clearGame() ล้างความจำและคะแนนของเกม
